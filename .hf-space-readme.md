# Face Emotion Classifier API

High-throughput Image Classification service using an ONNX-quantized Vision Transformer
([`trpakov/vit-face-expression`](https://huggingface.co/trpakov/vit-face-expression))
to classify 7 facial emotions: angry, disgust, fear, happy, neutral, sad, surprise.

## Endpoints

- **Web UI**: `/` — drag & drop a face photo, get prediction
- **API docs**: `/docs` (Swagger UI) and `/redoc`
- **Health check**: `GET /health`
- **Model info**: `GET /model-info`
- **Prediction**: `POST /predict` (multipart/form-data, field `file`)

## Quick test

```bash
curl -X POST -F "file=@your_photo.jpg" \
  https://YOUR-USERNAME-YOUR-SPACE.hf.space/predict
```

## Architecture

- **Model**: ViT-base, dynamically quantized to INT8 (~22 MB)
- **Runtime**: ONNX Runtime + FastAPI (async) + ProcessPoolExecutor for CPU-bound inference
- **Build**: Multi-stage Docker (FP32 deps stripped from final image)

Source code, CI/CD config and full project report: see GitHub repository.
